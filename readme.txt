Hide WordPress content behind a verification code to grow your WeChat official account followers.

== Description ==

LeHide is a WordPress plugin designed to help grow your WeChat official account followers. It allows you to hide content behind a verification code that users can obtain by following your WeChat official account. The plugin features a beautiful and practical interface.

**Key Features:**

* Hide WordPress content using shortcodes with verification code protection
* Per-post cookie verification that doesn't affect other content
* Cookie validity period of 7 days
* Easy-to-use editor button for inserting shortcodes
* Beautiful and user-friendly interface

**How it works:**

1. Wrap your content with the `[lehide]...[/lehide]` shortcode
2. Users see a QR code and verification form
3. Users follow your WeChat official account to get the verification code
4. After entering the correct code, hidden content is revealed
5. Verification status is saved in cookies for 7 days per post

For more information and installation guide, visit: https://www.lezaiyun.com/886.html

== Installation ==

1. Upload the LeHide folder to the `/wp-content/plugins/` directory
2. Activate the LeHide plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > LeHide Settings to configure the plugin
4. Set your WeChat QR code image, verification code, and module text
5. Use the editor button or manually insert `[lehide]...[/lehide]` shortcode in your posts
